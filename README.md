# dev-propinspector

A tiny FiveM dev-only prop inspector.

## How it works

- Equip `WEAPON_PISTOL`.
- Aim at a prop/object.
- The tool raycasts from your camera and shows the prop info.
- Press `E` while aimed at a prop to print the current target to F8.
- Press `F7` to toggle the inspector on/off.

## Install

1. Drop `dev-propinspector` into your resources folder.
2. Add this to server.cfg:

```cfg
ensure dev-propinspector
```

3. Restart the server or start the resource.
